# Invitación de Boda Elegante — Gualberto & Jhenny 💍

Este es el código fuente de una invitación de boda digital con estilo editorial minimalista (colores hueso y dorado). Está optimizada para cargar rápido, reproducir música de fondo y enviar la confirmación de asistencia directo a WhatsApp.

## Estructura de carpetas

```text
invitacion-boda/
├── index.html              ← Estructura de la página web
├── css/
│   ├── variables.css        ← 🎨 Colores principales (Hueso, Marfil, Dorado)
│   ├── reset.css            ← Estilos base (No tocar)
│   └── styles.css           ← Todos los estilos y animaciones
├── js/
│   ├── config.js            ← ✏️ EDITA AQUÍ Nombres, Textos y Fechas
│   └── main.js              ← Lógica y funcionalidad (reproductor, scroll)
└── assets/
    ├── images/
    │   └── couple/          ← Aquí colocas tus fotos (ej. portada.webp, foto-1.webp)
    └── music/
        └── cancion.mp3      ← Aquí colocas tu canción favorita
```
